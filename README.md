## 📄 Project Description

The **Employee Productivity Prediction System** is a Machine Learning based web application that predicts employee productivity using workplace operational data.

This project helps organizations analyze productivity patterns and make better decisions to improve workforce efficiency.

The system uses **Machine Learning with XGBoost** and is deployed using **Flask Web Framework**.

Users can input various workplace parameters and the model predicts productivity level instantly.

---

## 🎯 Objectives

- Predict employee productivity using ML
- Analyze workplace performance factors
- Provide real-time predictions through web interface
- Help organizations improve workforce efficiency
